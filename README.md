# PYTHON 3D Engine


## Made in the ****[Python Turtle](https://docs.python.org/3/library/turtle.html)** library 

### Runs on this one simple calculation:

```py
#Sets the x position to this
(0.5 * fov_val) * ((x + camera_X) / (z + 0.1 * camera_Z))

#Sets the y position to this
(0.5 * fov_val) * ((y + camera_Y) / (z + 0.1 * camera_Z))

#can be attached to the t.goto funtion using brackets - ()
```

## Requirements
* **[Python 3.11 or later](https://docs.python.org/release/3.11.0/)** -for main code
* **Keyboard** - for input
* **Turtle** - for input
